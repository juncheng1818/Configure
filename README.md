# 组态看板项目
### 自由拖动、组合，适用于工厂自定义看板或大屏
### JS框架：[Vite](https://vitejs.dev/) + [Vue3](https://vuejs.org/)
### UI框架：[Naive ui](https://www.naiveui.com/)
### 图形框架：[konva](https://konvajs.org/)
